# Ionicons Flutter Example

A demo app showcasing the 1,357 icons from Ionicons v8.0.13.

## Features

- Browse all icons in outline, filled, and sharp variants
- Search icons by name
- View icon details and codepoints

## Usage

```bash
flutter pub get
flutter run
```

## Using the Package

```dart
import 'package:ionicons_plus/ionicons_plus.dart';

Icon(Ionicons.add)
Icon(Ionicons.add_outline)
Icon(Ionicons.add_sharp)
```
